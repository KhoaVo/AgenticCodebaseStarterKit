# Agent Project Starter Pack

This starter pack gives you a reusable project skeleton for AI coding agents.

## Included

- `AGENTS.md`
- `memory-bank/` starter files
- `.clineignore`
- `.gitignore`

## Recommended usage

For a brand new project:

1. Copy these files into the repo root.
2. Commit them right away.
3. Tell your coding agent to read `AGENTS.md` first.
4. Keep the memory-bank files updated as work progresses.

For an existing repo:

- copy the files manually

## Starter structure

```text
AGENTS.md
memory-bank/
scripts/
.clineignore
.gitignore
README-STARTER.md
```
